Project Information

This project was intended to help users keep track of their tasks by allowing them to organize it into one place.


First you are greeting with an introductory page, where you can use the navigation bar above to choose the page you want to navigate to.


When you click the Home button, you are greeted by the introduction page.


When you click the Task List button, you go to the task list page.


When the Shop button is clicked, you go to the shop page - however, this is still currently a work in progress


Finally, the information page brings you to the page you are on right now


The biggest part of this project is, of course, the task list.
